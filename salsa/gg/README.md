# GitGud client

### Supported

- **✓** `gg clone`
- **✓** `gg status`
- **✓** `gg commit -m"<commit-message>"`
- **✓** `gg update <commit-id>`
- **✓** `gg amend`
- **✓** `gg evolve`
- **✓** `gg rebase`
- **✓** `gg sync`

### TODO / Known Issues

- **✗** `gg upload` not supported at all
- **✗** `gg snapshot` missing CLI support
- **✗** `gg sync` does not handle already merged commits
